# 强度削弱

## 功能

将"昂贵"的运算替换为等价但更快的运算。

## 支持的替换

### 乘法 → 移位

```
x * 2   →  x << 1
x * 4   →  x << 2
x * 8   →  x << 3
x * 16  →  x << 4
...（仅当乘数是2的幂时）
```

### 除法 → 移位

```
x / 2   →  x >> 1    （有符号：算术右移）
x / 4   →  x >> 2
x / 8   →  x >> 3
```

```
x udiv 2  →  x lshr 1  （无符号：逻辑右移）
x udiv 4  →  x lshr 2
```

### 取余 → 按位与

```
x urem 2   →  x & 1
x urem 4   →  x & 3
x urem 8   →  x & 7
x urem 16  →  x & 15
...（仅当除数是2的幂时，仅无符号）
```

## 性能收益

| 原始指令 | 替换为 | x86 延迟 | 加速 |
|---------|--------|---------|------|
| imul r, 4 | shl r, 2 | 3-4 cycles → 1 cycle | 3-4× |
| idiv r, 8 | sar r, 3 | 20-30 cycles → 1 cycle | 20-30× |
| div r, 16 | shr r, 4 + and | ~30 cycles → 2 cycles | 15× |
