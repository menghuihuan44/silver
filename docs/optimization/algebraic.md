# 代数简化

## 功能

利用代数恒等式消除冗余操作。

## 支持的恒等式

### 加法和减法

```
x + 0  →  x         0 + x  →  x
x - 0  →  x         x - x  →  0
```

### 乘法和除法

```
x * 1  →  x         1 * x  →  x
x * 0  →  0         x * -1 →  0 - x
x / 1  →  x
```

### 位操作

```
x & 0  →  0         x & x  →  x         x & -1 →  x
x | 0  →  x         x | x  →  x         x | -1 → -1
x ^ 0  →  x         x ^ x  →  0
x << 0 →  x         x >> 0 →  x
```

### 结合律

```
(x + c1) + c2  →  x + (c1 + c2)
(x - c1) - c2  →  x - (c1 + c2)
(x + c1) - c2  →  x + (c1 - c2)
```

## 与常量传播的协作

结合律优化将分散的常量合并，使常量传播能进一步折叠：

```
优化前: (x + 5) + 10
  结合律: x + 15
  常量传播: 如果 x 已知为 3 → 18
```
