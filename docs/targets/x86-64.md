# x86-64 目标平台

## 寄存器映射

Silver 使用统一的 `MachineRegister` 枚举，x86-64 映射到物理寄存器：

| Silver | x86-64 | 用途 |
|--------|--------|------|
| REG_RAX | RAX | 返回值 / 临时 |
| REG_RCX | RCX | 参数4 / 临时 |
| REG_RDX | RDX | 参数3 / 返回值高位 |
| REG_RBX | RBX | 被调用者保存 |
| REG_RSI | RSI | 参数2 |
| REG_RDI | RDI | 参数1 |
| REG_R8-R11 | R8-R11 | 参数5-6 / 临时 |
| REG_R12-R15 | R12-R15 | 被调用者保存 |
| REG_RSP | RSP | 栈指针 |
| REG_RBP | RBP | 帧指针 |
| REG_XMM0-XMM7 | XMM0-XMM7 | 浮点参数/临时 |
| REG_XMM8-XMM15 | XMM8-XMM15 | 浮点被调用者保存 |

## 调用约定

**System V AMD64 ABI**：

- 前6个整数参数：RDI, RSI, RDX, RCX, R8, R9
- 前8个浮点参数：XMM0-XMM7
- 返回值：RAX（RDX 用于 128 位返回值高位）
- 栈对齐：16 字节
- 红区：RSP 下方 128 字节

## 编码特点

- 变长编码：1-15 字节
- REX 前缀：标记 64 位操作和扩展寄存器
- ModRM 字节：寻址模式 + 寄存器编码
- SIB 字节：Scale-Index-Base 寻址
- SSE/SSE2 前缀：浮点指令

## 支持指令

- 所有整数算术和位操作（含立即数变体）
- 条件分支（Jcc）和 SETcc
- 间接调用/跳转
- LEA 地址计算
- IDIV/CDQ 除法序列
- SSE2 浮点：ADDSD, SUBSD, MULSD, DIVSD, UCOMISD
- 类型转换：CVTSI2SD, CVTSD2SI, MOVSXD
