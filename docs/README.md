# Silver 编译器后端 — 使用手册

**Silver** 是一个快速、自包含、跨平台的编译器后端。

## 核心特性

- **编译速度极快**：15–20 M IR/s 的吞吐量，峰值可达 20.5 M IR/s（在2.1GHz下）
- **代码质量好**：综合 22% 代码缩小，1.63× 估算加速
- **自包含**：不需要外部汇编器或链接器，可直接输出 ELF / PE / Mach-O
- **始终优化**：不区分 O0 / O1 / O2，始终运行完整的优化流水线
- **三平台支持**：x86-64、ARM64（AArch64）、RISC-V64
